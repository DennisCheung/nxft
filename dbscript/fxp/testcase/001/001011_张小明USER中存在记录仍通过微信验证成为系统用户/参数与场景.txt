使用 000\init_user.xml

参数：
  1、配置环境常量：
    [QYH_VISTOR_DEPT]    企业号通讯录部门 游客部门

    
       
  2、参数：
     1）微信企业号 userid= 234567 
 

场景：

  1、张小明通过微信企业号--企业小助手--用户注册成为 系统游客。 
  
  2、关闭“用户注册”界面
  
     1）自动验证：当前登录的微信号或微信号绑定的手机 等于 微信通讯录中的微信号或手机，当前微信和企业号通讯录关注。
     
        
     
     2）手动验证：（当前登录的微信号或微信号绑定的手机 <> 微信通讯录中的微信号或手机）
     
           点击“企业小助手” 应用，完成微信关注
           
        A、输入微信通讯录中的手机号及短信验证码，完成当前微信和和企业号通讯录绑定（关注）。
        B、输入微信通讯录中的邮箱号及邮箱验证码，完成当前微信和和企业号通讯录绑定（关注）。
        
        
   3、完成微信企业号关注后，获取张小明在企业号通讯录userid= 234567 调用本服务，成为系统用户
   
   4、自动验证/手工验证通过后提示：您已通过身份验证，加入企业号。

   

结果：

  1、判断该手机号是否注册过用户，如已注册，只实现微信通讯录绑定功能
    
      select count(*) from fxp.p_user t where t.fmobile = 输入手机号
      
      1.1 记录数 = 0  插用户表，fxp.p_user 同001010_张小明通过微信验证成为系统用户
  
      1.2 记录数 = 1  不插用户表，fxp.p_user
    
  2、select count(*) from fxp.p_qyh_user t where t.fmobile = 输入手机号
  
      2.1 记录数 = 0  微信企业号用户表 fxp.p_qyh_user 同001010_张小明通过微信验证成为系统用户
  
      1.2 记录数 = 1  不插微信企业号用户表 fxp.p_qyh_user
  
  
  3、删除fxp.p_qyh_guest   fqyh_user_id= 234567
